Instalar Octopress

Nov 1st, 2014 octopress

Instalar octopress

http://octopress.org/docs/setup/ 1. Primero instalamos Git sudo apt-get install git

  1. Instalamos Ruby. Debe ser una versión superior a la 1.93 sudo apt-get install ruby sudo apt-get install ruby-dev

  2. Instalamos dependencias: bundler sudo gem install bundler bundle install

Poner octopress en un nuevo ordenador

http://blog.zerosharp.com/clone-your-octopress-to-blog-from-two-places/ 1. Primero debemos tener SSH-Keys listas para acceder a github: https://help.github.com/articles/generating-ssh-keys/

  1. Vamos a nuestro repositorio de GitHub y miramos nuestra SSH clone URL. Probablemente sea algo así: git@github.com:Usuario/repositorio.git

  2. Clonamos el branch a nuestra carpeta local git clone git@github.com:Usuario/repositorio.git DirecciónCarpetaLocal
